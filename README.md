# statistical-dsp-studies

## Overview

This repo contains notebooks implementing statistical signal processing methods 

applied to biosignal and simulated data.  

Refer to individual notebook READMEs for details. 



## Setup

```bash

python -m venv venv

source venv/bin/activate

pip install -r requirements.txt

python -m ipykernel install --user --name=statistical-dsp-studies --display-name "statistical-dsp-studies"

```
